# Glossaire

Termes et concepts utilisés dans ShadowBot Touch.

## A

**AES-256**  
Algorithme de chiffrement symétrique utilisé pour sécuriser les mots de passe dans la base de données.

**AnkaBot**  
Bot référence pour Dofus Touch que ShadowBot vise à répliquer.

**API**  
Application Programming Interface - Interface de programmation permettant l'interaction avec le serveur.

**Auto-reconnexion**  
Fonctionnalité permettant au bot de se reconnecter automatiquement en cas de déconnexion.

## B

**Backoff exponentiel**  
Stratégie d'augmentation progressive du délai entre les tentatives de reconnexion.

**Bot**  
Instance automatisée gérant un compte Dofus Touch.

## C

**CRUD**  
Create, Read, Update, Delete - Opérations de base sur les données.

## D

**Dashboard**  
Interface principale d'administration de ShadowBot Touch.

**Dofus Touch**  
Version mobile du MMORPG Dofus.

## E

**Endpoint**  
Point d'accès de l'API REST (ex: `/api/accounts`).

## F

**Farm**  
Action de récolte automatique de ressources dans le jeu.

## H

**Heartbeat**  
Signal périodique envoyé pour maintenir la connexion active.

**HTTP/HTTPS**  
Protocole de transfert hypertexte (sécurisé).

## J

**JWT**  
JSON Web Token - Standard de tokens d'authentification.

## K

**Kamas**  
Monnaie virtuelle de Dofus.

## L

**Latence**  
Délai entre l'envoi d'une requête et la réception de la réponse (en millisecondes).

## M

**Middleware**  
Fonction intermédiaire dans le traitement des requêtes Express.

**mitmproxy**  
Outil de capture et d'analyse du trafic réseau HTTP/HTTPS.

## P

**Primus**  
Bibliothèque WebSocket wrapper utilisée par Dofus Touch.

**Proxy**  
Serveur intermédiaire masquant l'adresse IP réelle.

**PvP**  
Player versus Player - Combat entre joueurs.

## R

**Rate limiting**  
Limitation du nombre de requêtes par période de temps.

**REST**  
Representational State Transfer - Style d'architecture pour les APIs web.

## S

**SOCKS4/SOCKS5**  
Protocoles proxy de niveau socket, SOCKS5 supportant l'authentification.

**sql.js**  
Implémentation JavaScript de SQLite, sans compilation native.

**SQLite**  
Système de gestion de base de données embarquée.

## T

**Token**  
Jeton d'authentification temporaire.

## U

**Uptime**  
Pourcentage de temps pendant lequel un service est opérationnel.

## W

**WebSocket (WS/WSS)**  
Protocole de communication bidirectionnelle en temps réel (sécurisé avec WSS).
